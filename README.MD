# WP Coop Theme

Child theme boilerplate to be used with WP Coop Theme

## General

Please visit [WP Coop Theme](https://gitlab.com/coopdevs/wp-coop-theme) for more detailed documentation

## Development

* Run `npm install`
* Run `npm run watch` to start developing
